# COWILL AI LAB 電子名刺LP

和歌山を拠点に活動する AI講師・AIコンサルタント **東尾真紀子（まっきー）** の電子名刺LP。

- 方向性：**ナチュラル和モダン × 知的AIブルー**
- 軸：「AIは、人の力を引き出すための伴走者」
- 目的：売り込みではなく、「この人なら安心して相談できそう」と感じてもらう信頼の名刺

## 構成

スマホ縦1カラム・全9セクション
（① ファーストビュー ② 一言メッセージ ③ 想い ④ こんな方に ⑤ できること ⑥ 大切にしている考え方 ⑦ プロフィール ⑧ 活動・実績 ⑨ お問い合わせ）

設計の詳細は、社内docs（非公開）の `lp-structure.md` / `lp-design-policy.md` / `lp-canva-brief.md` に準拠。

## ファイル

| ファイル | 役割 |
|---------|------|
| `index.html` | LP本体 |
| `styles.css` | デザイン（配色・フォント・余白・レスポンシブ） |
| `images/` | 画像（hero.jpg / logo.png / profile.jpg / wakayama.jpg） |
| `.nojekyll` | GitHub Pages の余計な処理を止める |

## ローカルでの確認

`index.html` をブラウザで開く（ダブルクリック）だけで表示できます。

## 公開（GitHub Pages）※ URLが必要になったら

1. このフォルダの中だけで `git init`（親フォルダでは行わない）
2. 公開リポジトリ `cowill-lp` を作成し push
3. Settings → Pages → Source = main / (root)
4. 発行された URL をスマホで確認 → QRコード作成

## 安全メモ

- 公開リポジトリには **公開してよい情報だけ** を置く
- 個人情報・業務docs・APIキー・実在データは入れない
- 写真・実績・お客様の声は、掲載許可を確認してから掲載する
